# 坤坤的网站 - 前端项目

## 项目概述

坤坤的网站前端项目基于Vue 3 + Element Plus构建，采用现代化的组件化设计和响应式布局，提供优秀的用户体验。

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vue Router** - 官方路由管理器
- **Pinia** - 状态管理库
- **Element Plus** - Vue 3 UI组件库
- **Vite** - 下一代前端构建工具
- **Axios** - HTTP客户端

## 项目结构

```
src/
├── assets/           # 静态资源
│   ├── main.css      # 全局样式
│   └── base.css      # 基础样式
├── components/       # 公共组件
│   ├── TopNavbar.vue # 顶部导航栏
│   ├── BrandTag.vue  # 品牌标签
│   └── UserAvatar.vue # 用户头像
├── composables/      # 组合式函数
│   └── useAuth.js    # 用户认证逻辑
├── router/           # 路由配置
│   └── index.js      # 路由定义
├── stores/           # 状态管理
│   └── user.js       # 用户状态
├── utils/            # 工具函数
│   ├── request.js    # HTTP请求封装
│   └── common.js     # 通用工具函数
├── views/            # 页面组件
│   ├── Home.vue      # 首页
│   ├── Tools.vue     # 工具页
│   ├── Blog.vue      # 博客页
│   ├── Login.vue     # 登录页
│   └── Converter.vue # 转换工具页
├── App.vue           # 根组件
└── main.js           # 应用入口
```
